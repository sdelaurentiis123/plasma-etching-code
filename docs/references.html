Before formal use

Web-sourced entries (marked web) should have their exact authors, year, volume and pages verified against the primary venue before going into a thesis or paper. Classic entries (marked classic) are given with best-known bibliographic details.

A. Feature-scale topography tools & frameworks

  1. classic Quell et al. ViennaPS: A flexible framework for semiconductor process simulation. ViennaTools, 2025. — paper · code · docs. Our reference incumbent.
  2. web Riedel, X. GPU-Accelerated Ray-Tracing for Particle Simulations in ViennaPS. TU Wien, 2026. — thesis. OptiX engines incl. level-set voxel mesh.
  3. web Synopsys. Sentaurus Topography 3D. — page. Radiosity default; MC optional.
  4. Coventor / Lam Research. SEMulator3D. Monte Carlo, explicit surface; virtual fab.
